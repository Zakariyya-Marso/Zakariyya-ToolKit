What is Zakariyya's Toolkit?

A Technician's Swiss Army Knife

Zakariyya's Toolkit is a Live Operating System that runs entirely from your USB drive's RAM. Whether your computer won't boot, has been locked by ransomware, or you need to wipe a drive for a fresh start, this toolkit provides the environment you need.

It skips the "Standard Windows" limitations, giving you full access to system files, registry editors, and hardware diagnostics without ever needing to log in to the host PC.

Key Features:

  • Bypass Windows 11 Barriers: Auto-patch TPM and Secure Boot requirements.
  • Virus Cleaning: Scan drives from a clean environment where viruses can't hide.
  • Partition Magic: Resize, format, and rescue lost partitions easily.
  • Data Rescue: Copy files from a dead OS to an external drive.

Detailed Usage Guide

Follow these steps to successfully deploy the toolkit.

01

Prepare the USB Drive

Download Ventoy from the links below. Install it onto your USB drive. Once finished, simply copy the Zakariyya Toolkit ISO directly onto the USB drive. You can also copy your Windows 11 ISO to the same drive.

02

Booting the Environment

Insert the USB into the target PC. Restart the computer and tap your Boot Menu key (usually F12, F11, or ESC). Select your USB drive and choose "Zakariyya's Toolkit" from the list.

03

Installing Windows 11

Once inside the toolkit, locate your Windows 11 ISO. Right-click it and select Mount. Open WinNTSetup from the desktop. Point it to the install.wim file in the mounted drive usually found inside the source folder, select your C: partition, and click 'Setup'.
